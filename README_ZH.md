# X Video Downloader

Android Twitter/X 视频下载器

## 构建状态

[![Build APK](https://github.com/qq00150610-cpu/x-video-downloader/actions/workflows/build.yml/badge.svg)](https://github.com/qq00150610-cpu/x-video-downloader/actions/workflows/build.yml)

## 功能

- Twitter/X 视频下载
- 多画质选择 (SD/HD/2K/4K)
- 内置 ExoPlayer 播放器
- 本地视频管理
- Material Design 3 UI

## 技术栈

- Kotlin
- ExoPlayer (Media3)
- Room Database
- Coroutines + Flow
- Material Design 3

## 构建

```bash
./gradlew assembleDebug
```

APK 输出位置: `app/build/outputs/apk/debug/app-debug.apk`
